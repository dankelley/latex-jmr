About latex-jmr

Latex-jmr provides a LaTeX style-sheet for publishing in the Journal of Marine Research.

Downloading and using latex-jmr

To browse the source, visit http://github.com/dankelley/latex-jmr, or download it as a zip file, as a tar file. Git users may also clone it, as follows: git clone git://github.com/dankelley/latex-jmr.

Reporting errors or suggesting improvements

To report an error, or suggest an improvement, please open an issue.
