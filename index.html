About latex-jmr

latex-jmr provides a LaTeX style-sheet for publishing in the Journal of Marine Research.

Using latex-jmr

The package is normally installed from within R. To report errors in latex-jmr, or to suggest features, open an issue.

Developers

To browse the source, visit http://github.com/dankelley/latex-jmr. The source is also available as a zip file and a tar file, and as a git clone, accessed as follows: 

$ git clone git://github.com/dankelley/latex-jmr
